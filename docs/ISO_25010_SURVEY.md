# ISO/IEC 25010 Quality Evaluation Survey — PDAD Information System

## Instructions

Respondents rate each statement on a 5-point Likert scale:
1 = Strongly Disagree, 2 = Disagree, 3 = Neutral, 4 = Agree, 5 = Strongly Agree

---

## Section A: Functional Suitability

A1. The PWD ID validation feature correctly verifies my registered PWD ID.
A2. The employment type prediction gives a reasonable suggestion based on my profile.
A3. The job recommendations shown are relevant to my skills and preferences.
A4. The application submission process works as expected.

## Section B: Performance Efficiency

B1. Pages load within an acceptable time (under 3 seconds).
B2. Generating recommendations completes quickly (under 10 seconds).
B3. The system responds smoothly when I interact with it.

## Section C: Usability

C1. The interface is easy to understand.
C2. I can complete tasks without needing external help.
C3. The navigation menu is clear and organized.
C4. Error messages are helpful and clear.
C5. Text is readable (font size, contrast).

## Section D: Reliability

D1. The system works consistently without crashes.
D2. My data is saved reliably.
D3. When errors occur, the system recovers gracefully.

## Section E: Compatibility

E1. The system works on my preferred browser (Chrome, Firefox, Edge).
E2. The system works well on my device (desktop, tablet, mobile).

## Section F: Security

F1. My personal information feels secure.
F2. I feel confident that my PWD data is handled with privacy.

## Section G: Maintainability (developer respondents only)

G1. The codebase is easy to understand.
G2. Adding new features would not require extensive rework.

## Section H: Portability

H1. The system can be installed easily on a new environment.

## Open-ended Questions

1. What did you like most about the PDAD Information System?
2. What improvements would you suggest?
3. Were there any features that were confusing or difficult to use?
4. Would you recommend this system to others? Why or why not?
